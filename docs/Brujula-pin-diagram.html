Tierra común — todos los periféricos comparten GND
  • GND→ TFT, HMC5883L, LSM303, MS5837, SD
Wire (I2C, Core 1): HMC5883L (0x1E) + LSM303 Accel (0x18, SA0=GND) comparten IO1 (SDA) y IO2 (SCL). Definido en platformio.ini: I2C_SDA=1, I2C_SCL=2.
Wire1 (I2C, Core 0): MS5837-30BA en IO8 (SDA) e IO13 (SCL). Inicializado con Wire1.begin(8, 13). Bus exclusivo del Core 0 para evitar colisiones con Wire del Core 1. Si el sensor no se detecta al arrancar, depthTask usa el perfil simulado diveProfile().
SPI compartido (TFT + SD): IO3 (MOSI) e IO4 (SCLK) son usados tanto por el ST7735 como por el módulo MicroSD. El árbitro es el CS: TFT_CS=IO11, SD_CS=IO6. Inicializado con SPI.begin(4, 5, 3) (SCLK, MISO, MOSI).
